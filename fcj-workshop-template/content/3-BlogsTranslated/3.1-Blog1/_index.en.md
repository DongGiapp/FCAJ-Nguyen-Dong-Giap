# Optimizing LLM Operating Costs with Intelligent Prompt Routing

This blog explores Amazon Bedrock's **Intelligent Prompt Routing** — a serverless approach that dynamically routes prompts to the most cost-effective model in a family based on query complexity, avoiding the "single-model trap" of using premium models for every task.

Key topics covered in the Vietnamese translation:
- Technical mechanism of the Prompt Router
- Quantitative cost analysis (Haiku vs Sonnet)
- Applications for Model Agent architectures
- System limitations (language support, customization)
